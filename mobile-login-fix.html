常见问题与解决方案

  • 弹窗被阻止

    Chrome 移动端默认阻止弹窗，导致 Google 登录窗口无法打开

  • 跨域限制

    CORS 策略阻止了第三方登录请求

  • Cookie/存储限制

    移动浏览器的隐私设置可能阻止必要的 Cookie

  • 网络连接问题

    不稳定的网络连接导致认证流程中断

    • 修复步骤

    清除浏览器缓存

    Chrome菜单 → 设置 → 隐私设置和安全性 → 清除浏览数据

    允许弹窗

    访问网站时，点击地址栏的锁图标 → 弹出式窗口和重定向 → 允许

    启用第三方Cookie

    Chrome设置 → 隐私设置和安全性 → Cookie和其他网站数据 → 允许所有Cookie

    检查网络连接

    确保网络稳定，尝试切换到WiFi或4G网络

    使用重定向登录

    如果弹窗登录失败，系统会自动切换到页面重定向登录方式
